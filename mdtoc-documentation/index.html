Getting Started

Get the Project

First clone the git repository :
$ git clone https://github.com/Teskann/mdtoc
Then move to the project folder :
$ cd ./mdtoc

Install Modules

This project uses the following non-standard modules :
  • marko
  • mdtoc
  • setuptools
If you don't have them, you can install these packages running pip :
$ pip install
$ pip install marko
$ pip install mdtoc
$ pip install setuptools
