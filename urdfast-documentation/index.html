Getting Started

Get the Project

First clone the git repository :
$ git clone https://github.com/Teskann/URDFast
Then move to the project folder :
$ cd ./URDFast

Install Modules

This project uses the following non-standard modules :
  • anytree
  • numpy
  • PyQt5
  • sympy
If you don't have them, you can install these packages running pip :
$ pip install anytree
$ pip install numpy
$ pip install PyQt5
$ pip install sympy
